Custom pipeline tools and automation for VFX production

Category Pipeline & Tools
Languages Python, C
Platforms Maya, Houdini, Nuke, Qt Designer, GitHub

Development of custom tools and scripts to streamline VFX production workflows. From lighting management utilities to asset pipeline automation, these tools improve efficiency and consistency across projects.

Light Finder Plus Custom UI tool designed to publish and load lighting setups, and keep track of versions efficiently within Maya. Created with Qt and Python.

Features:

  • Publish and load lighting setups
  • Version tracking
  • User-friendly interface
Download Tool
Light Manager Plus Custom UI tool designed to improve Maya's default lighting spreadsheet, allowing users to manage lighting setups and edit multiple lights parameters within Maya and Arnold. Created with Python.

Features:

  • Edit multiple lights parameters.
  • Quick light creation and utility shortcuts.
  • Randomize light parameters for look development and animations.
  • Scatter lights and create light rigs.
  • User-friendly interface
